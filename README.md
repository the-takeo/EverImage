EverImage
=========
### About this

指定したWebページ上の画像を取得し、ローカルにダウンロードするか、Evernoteに送信します。 
   
ログイン機能については、下記を使わせて頂いています。   
https://github.com/matchy2/EvernoteOAuth

### Function

・Webページ上の画像をEvernoteに送信   
・Webページ上の画像をローカルにダウンロード
